Na pasta 'Arduino' estão as bibliotecas usadas no desenvolvimento do jogo com as suas alterações feitas.
Na pasta 'game' estão os ficheiros do jogo principais:
	- 'game.ino' é onde se encontra o código com as mecânicas do jogo.
	- No ficheiro 'sprites.c' estão todos os gráficos usados no jogo em forma de 'bitmaps'.
	- O ficheiro python 'arduboy_screen.py' é usado para mostrar o ecrã da consola numa janela do computador. Este programa é usado principalmente para tirar capturas de ecrã.
